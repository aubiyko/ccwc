# ccwc

ccwc (short for [Coding Challenges](https://codingchallenges.fyi) word count) - [Write Your Own wc Tool challenge](https://codingchallenges.fyi/challenges/challenge-wc).

TL/DR: wc - word, line, character and byte count.
